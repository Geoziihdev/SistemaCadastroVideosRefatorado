# SistemaCadastroVideosRefatorado

Projeto refatorado do sistema de cadastro de vídeos desenvolvido em Java, com aplicação de princípios SOLID e separação de responsabilidades (MVC).

##  Descrição

Este projeto é a evolução de um sistema desktop desenvolvido anteriormente, que passou por um processo de refatoração com o objetivo de:

- Melhorar a organização do código
- Aplicar boas práticas de programação orientada a objetos
- Separar responsabilidades entre as camadas do sistema
- Preparar o sistema para futura migração para ambiente web

## Estrutura do Projeto

O projeto está organizado nas seguintes camadas:

- **model** → Representação dos dados (entidades)
- **SistemaDAO** → Acesso ao banco de dados
- **SistemaController** → Regras de negócio
- **SistemaView** → Interface gráfica (Swing)

## Conceitos aplicados

- Princípios SOLID (principalmente SRP)
- Separação de responsabilidades
- Padrão MVC
- Refatoração de código

## Testes

Foram realizados testes utilizando:

- Método `main()`
- Interface gráfica do sistema

## Tecnologias utilizadas

- Java
- Swing (interface)
- JDBC
- MySQL
